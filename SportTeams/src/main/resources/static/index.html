Получить всех игроков из базы данных

после перехода вы получите вывод всех игроков из базы в формате JSON.

можно сделать выборку по конкретной команде и по типу игрока

(Нападающий/Полузащитник/Защитник/Вратарь) нужно указать

варианты type=forward, midfielder, defender, goalkeeper

для этого нужно добавить параметры запроса team= и type= через &

например ?type=forward&team=spartak получим нападающих игроков спартака.

регистр ввода не имеет значения.

для получения отдельной записи по номеру в базе данных, можно добавить номер через слеш

например players/4 вернет запись с ID=4

Получить все команды

переход на страницу со списком всех команд,

для получения записей команд из базы данных по виду спорта

нужно добавить свойство sport=football, basketball, volleyball, hockey, baseball

для получения отдельной, можно добавить имя или номер через слеш

например http://localhost:8080/teams/3 вернет команду с ID=3,

или http://localhost:8080/teams/csk вернет запись команды ЦСК

есть возможность получить команды за период по дате основания

например http://localhost:8080/teams?fromdate=1900-01-01&tilldate=1999-12-31

выдаст всех за прошлый век. формат даты ГГГГ-ММ-ДД
